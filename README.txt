Setup:
1. Make sure 'blackjack.rb' and 'blackjack_tests.rb' are saved in a common directory
2. Open a bash window and navigate to the directory where your Ruby
		files are stored
		
Play Black Jack:
1. Run 'blackjack.rb' from your shell with no arguments by typing 'ruby blackjack.rb'
2. Follow the instructions to play a round of blackjack
	
Run Unit Tests:
1. Run 'blackjack_tests.rb' from your shell with no arguments by typing
		'ruby blackjack_tests.rb'
2. You will have to play a round of blackjack first, when the program terminates the
		test results will be displayed
